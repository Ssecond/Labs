Аргументы в команде разделяются пробелами.
Для ввода аргументов, состоящих из нескольких слов используются кавычки ("").
Обозначения.
 {} - обязательные аргументы.
 [] - опционально.
Команды.
 list - выводит все записи с телефонной книги;
 add {имя} {фамилия} {отчество} {номер телефона} {email} [заблокировать ли запись] - добавляет новую запись в телефонную книгу;
 delete {ID} - удаляет запись в телефонной книге по идентификационному номеру (ID);
 change {ID} {название параметра для изменения} {новое значение}.
